# Graphs and Trees

This directory is for practicing problems related to Graphs and Trees. These are heavily tested in Google interviews.

## Core Patterns to Practice:
1. **Depth First Search (DFS):** Both recursive and iterative. Crucial for tree traversals, counting islands, and exhaustively searching a space.
2. **Breadth First Search (BFS):** Crucial for finding the shortest path in unweighted graphs or level-order traversals in trees.
3. **Topological Sort:** For DAGs (Directed Acyclic Graphs), usually used for scheduling problems (e.g., Course Schedule).
4. **Union Find (Disjoint Set):** For finding connected components or cycle detection in undirected graphs.
5. **Dijkstra's Algorithm:** Shortest path in weighted graphs.
6. **Trie (Prefix Tree):** For string manipulation, autocomplete, and dictionary queries.
