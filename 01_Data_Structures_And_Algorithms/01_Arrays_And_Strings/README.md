# Arrays and Strings

This directory is for practicing problems related to Arrays and Strings. These are foundational for all coding interviews.

## Core Patterns to Practice:
1. **Two Pointers:** Important for sorted arrays or when you need to find a pair of elements.
2. **Sliding Window:** Used for problems asking for contiguous subarrays or substrings (e.g., "longest substring without repeating characters").
3. **Prefix Sum:** Useful when you need to calculate the sum of elements in a range multiple times O(1).
4. **Fast and Slow Pointers (Floyd's Cycle Finding):** Detecting cycles in linked lists, but also applicable to array problems like "Find the Duplicate Number".
